无开发者账号打包生产环境ipa包的方法
  • 在Build Settings-Build Active Architecture Only 中设置成NO Debug 和 Realse都要
  • 在Xcode描述文件配置选择自动。也就是设置General->automatic编译选项打勾，并添加一个免费个人账号（任何appleid都可以）。clean项目后然后编译项目。找到app格式的文件，一般在Product文件夹中
  • 创建一个Payload文件夹，将这个app文件放入后压缩变成zip
  • 将zip改成ipa格式
    这个ipa格式的包就是签名需要用到的包
